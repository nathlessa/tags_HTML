1. Introdução

Nessa página iremos abordar sobre algumas Tags HTML que aprendemos durante o módulo de Primeiros Passos em HTML do Curso Formação HTML Web Developer da Dio.

Tags HTML são instruções que informam ao navegador como formatar o texto. Essas tags estão presentes no HTML (Hypertext Markup Language) de todas as páginas da internet. Os servidores leem o código HTML para entender e renderizar o conteúdo corretamente. Essa leitura é feita de cima para baixo, da mesma forma que você está lendo este artigo agora.

2. Tags HTML

Tags de Texto

<h1> a <h6>

Representa a hierarquia dos títulos de uma seção, sendo <h1> o nível mais alto e <h6> o nível mais baixo.

<p>

Representa um parágrafo.

<mark>

Realça um texto específico.

<small>

Deixa o texto com uma fonte menor em relação ao texto circundante. Exemplo:
Não sonhe sua vida, viva seu sonho.

<blockquote>

Indica que o texto é uma citação longa. Exemplo:

"Você já percorreu um incrível percurso de aprendizado, demonstrando todo o seu esforço, dedicação e persistência. O conhecimento abre portas para um futuro brilhante. Eu acredito em você e estou aqui para te incentivar a seguir sempre em frente. A sua luta valerá a pena e você será capaz de conquistar coisas incríveis. Não desperdice o seu potencial e lembre-se que desistir não te levará a lugar algum. Os estudos, sim. Mesmo quando os desafios surgirem, não desanime. Você tem o conhecimento e a inteligência necessários para ser o seu suporte."

<strong>

Representa forte ênfase em uma parte importante do texto (negrito).

<i>

Representa um texto em formato itálico.

<u>

Representa um texto em formato itálico.

<sub>

Representa o texto subscrito embutido. Exemplo:
H2O

<sup>

Representa texto sobrescrito embutido. Exemplo:
22

<abbr>

Descreve uma abreviatura ou acrônimo.

<del>

Representa parte do texto que foi excluída de um documento da web.

Exemplo de uso da tag del :)

Tags de Lista

< ol >

Lista ordenada.

<ul>

Lista não ordenada.

<li>

Define um item de uma lista, seja ordenada < ol >, seja não ordenada. < ul >

Tags de Link

< a>

Cria um hyperlink (link em documentos da web internos/externos)
